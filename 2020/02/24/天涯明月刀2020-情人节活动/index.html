天涯明月刀2020 情人节活动

Posted on 2020-02-24 In Python Views:

天涯明月刀2020年情人节“天涯不远，明月尘缘”活动分析。因其中有概率问题，感兴趣就写了代码。运行一万次，几万次得到分布来推断冲多少钱可以得到多少币，也可以测自己有多黑。

活动介绍

每次活动的本质其实是一个抽奖活动。

充值 获得 尘缘令（抽奖次数），有概率可以抽中 幸运数字（大奖），大奖将获得额外奖励。

大奖也有保底机制，例如第一个幸运数字在抽第25次是会必定出现。

总共有五个大奖，保底次数和概率如下图所示

抽中幸运数字后的奖励为：

第一档：40币

第二档：60币

第三档：100币

第四档：160币

第五档：240币

充值获得抽奖次数的转换为：5元可得1枚尘缘令，50元得11枚尘缘令。

活动分析

至此我认为使用二元分布只分析中或者不中幸运数字便能解决这个问题。

python numpy库中的 np.random.binomial() 可以帮助我们。

例对于第一个幸运数字，概率为4%，并且会在第25次必中，因此我们用：

np.random.binomial(1,0.04,25)

表示每次抽取一次，抽25次，其中得到1表示抽中幸运数字。它将返回一个length为25的list如下图：

这时表示我们在第六次抽奖时抽中了幸运数字。那我们即可使用break跳出循环，进行下一阶段抽奖。

当然这只是一次的抽奖结果，在最后的代码中，我们将运行10000次来得到具有统计意义的结果。

更进一步
